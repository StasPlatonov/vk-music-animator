# Music Animator for VK (vkontakte, vk.com)

This extension add graphical visualizer to vkontakte audio player.
You can use different effects and themes while listening the music on web site vk.com.
Make your music fun!

New visual effects are coming soon!

You can find this extension on Chrome Web Store
https://chrome.google.com/webstore/detail/vk-music-animator/kningkodbgfoppmglhnbfllamkopmllc

You can find some instructions and screenshots here:
